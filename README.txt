GmailBot's Requirements: 
- Install Python on your computer, please make sure the python version is 3.7 or higher, (and Don't forget to check "Add PATH in windows" option during the installation).
- click "install.bat"
- add your emails to "profiles.csv"
- make sure that your emails have a firefox profile created before on your PC.
- make sure to enable "Shortcut option" from gmail settings for each profile (important: without this feature enabled NOTHING's GANNA WORK). 
- finally just Click on "start.bat" and the GmailBot will do the rest :D
